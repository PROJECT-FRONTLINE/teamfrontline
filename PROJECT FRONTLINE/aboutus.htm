About Us Page

This is a project is a part of IBM Call for Code COVID-19 Challenge.The main aim of this project is to serve our First responders who are always on the Frontline.

We believe that "First Responders need First Response"


For this purpose we have created a Frontline Chatbot to answer questions of the First Responders that they encounter at the Frontline.For this purpose we have used Watson Assistant Chatbot Technology.


We have also designed a form which serves as a data collection method from the First Responders regsrding their needs and the problems they faces on the Frontline. This data that we have collected is then fed into Watson Studio in order to produce Insights in terms of Data Visualization.
